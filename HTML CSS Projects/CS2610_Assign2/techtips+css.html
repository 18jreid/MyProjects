HTML Elements Catalog

<html>

  • purpose: HTML document root element; must be unique in document
  • display: N/A
  • contents:
    1. <head>
    2. <body>

<head>

  • purpose: container for metadata (data about data)
  • display: N/A
  • contents: metadata

<title>

  • purpose: defines the title of the document.
  • display: top of webpage tab
  • contents: title

<body>

  • name: idem.
  • purpose: Defines the documents body
  • display: Holds all structure and information of document
  • contents: contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

<h1> to <h6>

  • purpose: used to define HTML headings
  • display: desired image
  • contents: none

<p>

  • purpose: used to contain paragraphs
  • display: Holds all paragraph content in a contained box
  • contents: any desired information

<strong>

  • purpose: turns text bold
  • display: bolded text
  • contents: desired text to be bolded

<div>

  • purpose: divides html tags into a section
  • display: can be used to change margin, text, background-color, etc. of entire divisions
  • contents: anything inside the tag

<img>

  • purpose: used to embed an image in an HTML page
  • display: top of webpage tab
  • contents: title

<ul>

  • purpose: an unordered (bulleted) list
  • display: framework for bulleted list items
  • contents: contains the <li> tag

<ol>

  • purpose: defines an ordered list
  • display: framework for numbered list items
  • contents: desired information

<li>

  • purpose: defines a list item
  • display: either numbered or bulleted list items depending on the list type
  • contents: desired information
CSS Selectors Sheet

element

  • description: the element selector selects HTML elements based on the element name
  • purpose: to change any characteristic of all specific elements

class

  • description: The class selector selects HTML elements with a specific class attribute
  • purpose: to change any characteristic of any specific element with the class type

id

  • description: The id selector uses the id attribute of an HTML element to select a specific element
  • purpose: to change any characteristic of all any specific element with the id type
CSS Properties Sheet

margin

  • description: the space around an element
  • purpose: sets all the margin properties in one declaration

background-color

  • description: the background color of the element
  • purpose: changes the background color of an element

border-radius

  • description: a shorthand property for the four border-*-radius properties
  • purpose: changes the radius of an image

text-align

  • description: the horizontal alignment of where the text is
  • purpose: changes the horizontal alignment of text

font-size

  • description: the specific size of the font
  • purpose: changes the font size

color

  • description: the color of the text
  • purpose: changes the color of the text

font-family

  • description: the font of the text
  • purpose: changes the font of the text

text-decoration

  • description: specifies the decoration added to text
  • purpose: changes text decoration

text-indent

  • description: specifies the indentation of the first line in a text-block
  • purpose: changes the indentation

text-shadow

  • description: enables user to add shadow behind text
  • purpose: adds shadow to text
